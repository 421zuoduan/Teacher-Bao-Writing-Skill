# Teacher-Bao-Writing-Skill

Logic-first writing skill for AI and large-model research papers.

## What it helps with

- Drafting and revising abstracts, introductions, related work, methods, validation sections, and conclusions
- Writing partial outputs such as titles, contribution bullets, figure captions, and paragraph-level rewrites
- Translating or polishing paper text while keeping the original meaning and scope
- Keeping the paper logic clear, the prose restrained, and the paragraph-level flow nonredundant

## Contents

- `SKILL.md`: main skill instructions
- `agents/openai.yaml`: UI metadata
- `references/`: style and section guidance

## Use

Install or copy this folder as a Codex skill package.
